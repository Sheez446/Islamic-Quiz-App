# My First Quiz App

This is a simple **Islamic Knowledge Quiz App** built with HTML, CSS, and JavaScript.

## Features
- Multiple categories: Pillars of Islam, Prophets, Quran Knowledge
- Countdown timer for each question
- Score tracking with high score
- Images for welcome banner, quiz icon, and result icon
- Exit button added in JS

## How to Run
1. Clone this repo:
git clone https://github.com/yourusername/my-first-quiz-app.git
2. Open project folder
3. Open `index.html` in your browser
4. Start the quiz!
